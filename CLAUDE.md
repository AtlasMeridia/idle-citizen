# Idle Citizen

Autonomous sessions that use unused Claude Max plan quota. The launcher calls you at timed intervals after checking remaining quota.

## What To Do

1. Find the activity folder in `activity/` (any folder with a README.md)
2. Follow the instructions in that README

## Directory Structure

```
idle-citizen/
├── CLAUDE.md                    # This file
├── activity/                    # Activity folder(s)
│   └── {name}/README.md         # Activity instructions
└── app support/
    ├── logs/                    # Auto-generated session logs
    └── scripts/                 # Launcher scripts
```
